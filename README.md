 # EventPro Frontend

Event management platform — frontend codebase.
Built by Group 8, TechCrush Capstone 2026.

## Screens completed
- Inline Validation Component
- Sign Up Page
- Email Verification Screen

## Tech stack
- HTML, CSS, Vanilla JavaScript
- Mobile first — rem units, media queries
- React coming in Sprint 2

## Folder structure
- assets/      — images and icons
- components/  — reusable HTML components
- js/          — all JavaScript files
- pages/       — all HTML screens
- styles/      — all CSS files
